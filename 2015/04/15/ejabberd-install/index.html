ejabberd install 

1.安装依赖
brew install m4
brew install autoconf
brew install automake 
brew install libtool
brew install pkg-config

2.配置
cd xxxx/xxx/ejabberd
./autogen.sh
./configure
make

3.安装
sudo make install

4.启动
ejabberdctl start
ejabberdctl status

5.新建帐号
ejabberdctl register admin1 example.org FgT5bk3

然后编辑ejabberd.yml文件
acl:
  admin:
    user:
      - "admin1": "example.org"
access:
  configure:
    admin: allow

然后重起服务
ejabberdctl restart


5.打开管理后台
http://localhost:5280/admin/
用户名:admin1  密码:FgT5bk3
