InterlliJ Use

快捷键

注意 下面罗列的是一些常用快捷键:

快捷键 注释
command+P 方法参数提示
command+alt+L 格式化代码
command+alt+I 自动缩进，不用多次使用tab或着backspace键，也是比较方便的
ctrl+shift+space 代码补全，这个会判断可能用到的，这个代码补全和代码提示是不一样的
command+/ 注释一行或着多行 //
ctrl+shift+/ 注释/…/
shift+f6 重命名
command + F12 类似eclipse 的ctrl + O的功能outline

rebar相关

rebar 是一个遵循 Erlang/OTP 原则的 Erlang 项目构建工具，使用它可以减少构建标准 Erlang/OTP 项目架构配置的工作量，并且可以很容易的编译、测试、发布 Erlang 应用程序。更强大的是，rebar 提供一种依赖管理机制，它可以使开发者很方便地通过 Git、Hg 等方式重用常见的第三方 Erlang 模块或库。
